# aivibedigest-site

Статический сайт генерируется из одного шаблона и файла с выпусками.

- Данные выпусков: `data/issues.js`
- Генератор: `scripts/build-site.js`
- Сборка: `npm run build`
- Опционально для выпуска можно передавать `telegramPostUrl` и тогда на странице дайджеста появится ссылка на конкретный пост в Telegram
