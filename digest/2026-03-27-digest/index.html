Подробности по блокам

ИИ-ассистенты и автоматизация

Machinelearning рассказывает, как Claude оказался в списке топ-контрибьюторов репозитория OpenAI — участники конкурса Parameter Golf использовали его для генерации кода. Это показывает, что ИИ-ассистенты уже стали настолько обыденным инструментом, что их вклад путают с человеческим.

Machinelearning представляет шаблон Claude Code Game Studios, который организует 48 ИИ-агентов в полноценную игровую студию с иерархией директоров и специалистов. Система работает с Godot, Unity и Unreal Engine, координируя задачи от дизайна до программирования, но финальное решение всегда остаётся за человеком.

Мысли вслух делится опытом использования Claude Code как «агента с руками», который самостоятельно читает, редактирует файлы и выполняет задачи в терминале. В отличие от чат-ботов, он работает автономно, что полезно для обработки документов или переводов без постоянного вмешательства.

По информации AI и грабли, владельцы сервисных компаний часто хотят упаковать свою экспертизу в ИИ-продукт, но автор советует сначала автоматизировать внутренние процессы. Это позволяет той же командой выполнять в 3-5 раз больше заказов, наращивая маржу, а не продавая продукт со скидкой.

Модели и платформы

Machinelearning сообщает об обновлении инструмента File Search в Yandex AI Studio: теперь он работает с видео, аудио и таблицами, позволяя ИИ-агентам извлекать смысл из мультимедийных источников. Это снижает порог входа для бизнес-кейсов, где данные хранятся в разных форматах.

Machinelearning собрал несколько важных анонсов: Google представила голосовую модель Gemini 3.1 Flash Live для плавного диалога, Mistral выпустила открытую TTS-модель Voxtral для клонирования голоса, а Cohere — точную модель распознавания речи Transcribe. Также Intel анонсировала видеокарты Arc Pro B70/B65 с 32 ГБ памяти для инференса нейросетей по более доступной цене.

Machinelearning описывает концепт DeepMind — браузер-генератор сайтов на базе Gemini 3.1 Flash-Lite, который создаёт HTML и CSS на лету вместо загрузки страниц. Пока система склонна к ошибкам и тратит много токенов, но это шаг к автономным агентам, которым нужно быстро собрать интерфейс.

Исследования и инструменты

Data Secrets рассказывает о TRIBE v2 от Meta — открытом симуляторе человеческого мозга, который предсказывает нейронную активность точнее, чем индивидуальный МРТ. Модель объясняет до 80% сигнала в отдельных областях, и её точность будет расти с масштабированием.

gonzo-обзоры ML статей разбирает фреймворк HyperAgents, где агент задачи и метаоптимизирующий агент объединены в самореферентную программу. Система автономно переписывает свою логику и методы оптимизации, что позволяет изобретать переносимые решения для разных доменов, например, дизайна наград для роботов.

Старший Авгур сообщает, что Percepta выложила в открытый доступ код компилятора C в трансформеры. Это позволяет встраивать инструменты напрямую в веса моделей, открывая новые возможности для интеграции вычислений в нейросети.
