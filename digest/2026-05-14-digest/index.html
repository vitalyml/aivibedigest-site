Подробности по блокам

Модели и бенчмарки

Machinelearning сообщает, что GPT 5.5 в режимах high и xhigh впервые полностью решила задачу cmatrix из ProgramBench — ни одна публичная модель раньше не доводила такие задания до конца.

Условия для разработчиков

эйай ньюз и Machinelearning анонсировали, что с 15 июня подписчики Claude получат кредиты, равные сумме подписки, для SDK, CLI и сторонних приложений — свой бюджет не затронет основной лимит.

Machinelearning рассказал о трёхкратном росте расходов на Claude у IT-директора Telaid — корпоративных клиентов переводят на оплату по фактическому использованию, но компании продолжают платить.

Claude Code: агенты в деле

эйай ньюз и Анализ данных описали новые функции Claude Code: Agent View позволяет видеть все сессии агентов в одном окне, а режим /goal заставляет модель работать до результата без постоянных остановок.

Сиолошная разобрала кейс переписывания Bun на Rust с помощью Claude Code — за 10 дней агенты в цикле сгенерировали более миллиона строк кода, бинарник стал легче, а утечки памяти исчезли.

Анализ данных поделился историей о восстановлении доступа к биткоин-кошельку с 5 BTC — Claude перебрал 3,5 триллиона вариантов пароля, проанализировал старые файлы и сопоставил wallet.dat с найденной seed-фразой.

Love. Death. Transformers. показал, как skill на базе Claude создал модель deepseek v4 на 100 миллионов параметров, обучил её на TinyStories и выложил демо-пространство.

Инфраструктура и инструменты

Анализ данных объявил о выходе PyTorch 2.12 — batched linalg.eigh ускорился до 100x, появился torch.accelerator.Graph для универсальной работы с ускорителями, улучшена поддержка ROCm и distributed training.

Анализ данных представил Portable-AI-USB — флешка с Ollama, AnythingLLM и локальной моделью даёт полностью офлайнового AI-ассистента без облака и регистрации.
